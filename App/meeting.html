Mes compétences

Je maîtrise les dernières technologies pour créer des sites web performants et professionnels. Toujours à jour pour garantir un rendement optimal. Actuellement entrain d'apprendre des frameworks tels que React JS et Vue JS.

Linux

Linux est un système d'exploitation souvent utilisé pour les serveurs. Il offre une stabilité, fiabilité et flexibilité

Git

Git est un système de versionnage distribué pour le développement facilitant le travail en collaboration et la gestion des versions

PHP

PHP est un langage de programmation polyvalent pour les sites web, les applications web et d'autres utilisations

C

C est un langage de programmation performant pour les systèmes d'exploitation et les applications, avec une syntaxe concise

MySQL

MySQL est un système de base de données open source largement utilisé, connu pour sa fiabilité et sa facilité d'utilisation.

JavaScript

Le JavaScript permet de rendre le site dynamique et d'intégrer des animations ou des actions automatisées

HTML

L'utilisation d'HTML permet d'imbriquer un site de façon sémantique et de le rendre optimisé pour le SEO

CSS

Le CSS permet de changer l'esthétique et d'organiser un site le rendant agréable à l'oeil sur tout les appareils
